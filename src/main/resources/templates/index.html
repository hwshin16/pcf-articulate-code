This is Articulate! 1111

이 애플리케이션의 목적은 Pivotal Cloud Foundry 플랫폼, 구체적으로 애플리케이션의 워크로드를 구동하는 Elastic Runtime의 개념과 기능을 분명히 하는데 있습니다.

애플리케이션 구조

articulate는 친숙하고 탐색 가능한 사용자 인ㅌ페이스를 제공하는 웹 애플리케이션입니다. 그러나 데이터를 직접 다루지는 않습니다. 이것은 데이터를 관리하는 attendee-service 응용 프로그램에 따라 다릅니다. attendee-service는 데이터를 MySQL 데이터베이스에 보관합니다.

이 애플리케이션을 사용하는 방법

위의 각 메뉴 항목은 플랫폼이 제공하는 일련의 기능을 보여주는데 도움이 되는 페이지로 연결됩니다. 마지막 항목인 Spring Boot는 Spring Boot와 함께 제공되는 기능을 강조 표시되며, 몇 분 안에 생산 준비가 완료된 마이크로 서비스를 구축하는 데 도움을 줍니다.

각 페이지는 각 페이지의 레이아웃은 Accordion 컨트롤과 최대 3 개의 그룹으로 구성되는 동일한 레이아웃을 가집니다:

  1. 애플리케이션 환경 정보 - PCF에서 구동되는 애플리케이션의 환경에 대한 정보를 제공합니다. 애플리케이션명, 컨테이녀 및 서비스 정보를 볼 수 있습니다. 로드밸런싱, 자가 치유 서비스 바인딩 같은 것을 보여주는데 유용합니다.
  2. 설명 - 주어진 페이지에 대한 부가 정보
  3. The Twelve-Factor App - 현대적이고 확장 가능한 애플리케이션을 구축하는 방법론입니다. 적용 가능한 factor에 대한 링크가 제공됩니다.
